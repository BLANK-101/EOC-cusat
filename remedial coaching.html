WEB PAGE DESIGNING

Activity 1 : create a web page similar to http://ugcsct.cusat.ac.in and upload the files into a github repository for evaluation .

  • Create an html file with general tags.

  • Insert a navbar ( copy codes for navbar from getbootstrap.com ) similar to the reference web page .

  • Insert a logo image and text similar to the reference web page .

  • Insert an image gallery ( you may copy codes for carousel from getbootstrap.com ) similar to the reference web page .

  • Copy the content from the reference web page and customize it .

  • Insert a footer similar to the reference web page .

Activity 2 : create two more web pages with similar theme ( logo ,navbar and footer ) and upload the files into the same github repository for evaluation .

  • One of the links in the navbar should open these new webpages

  • Insert a form for student details ( copy codes for form from getbootstrap.com )in one of the web page .

  • Create a simple tutorial containing procedures for web page creation ( based on finished activites of this course ) in the other web page .
